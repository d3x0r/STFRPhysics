Physics Note

Σ GEODESIC mode computes acceleration from the effective coordinate geometry of displaced transport space:

ds² = −c²/(1+Σ)² dt² + (1+2Σ) dr² + r² dΩ²

The path cost at each point is set by the local cumulative displacement Σ(r) = GM/(c²r). Precession emerges as a single geometric result — not from an added force term.

SCHWARZSCHILD mode uses the old αGMh²/(r⁴c²) force correction for comparison.
